# Introduction to FlexoraCSS

Flexora is a utility-first framework that mimics the high-end look of professional SaaS products. 


## Core Principles
1. **Speed:** Under 10kb gzipped.
2. **Design:** Modern Indigo color palette.
3. **Flexibility:** Powered by CSS Variables.

| :--- | :--- |

| Grid | Full |

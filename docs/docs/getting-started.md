#  Getting Started

Welcome to **FlexoraCSS** — a lightweight, utility-first UI framework.

Build fast, responsive interfaces using simple utility classes.

---

##  Installation

### CDN (Quick Start)

Add this to your HTML:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/codegear-2011/flexora-css/flexora.css">
```
```js
<script src="https://cdn.jsdelivr.net/gh/codegear-2011/flexora-css/flexora.js">
```
